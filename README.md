# Dockerized Microservices Architecture

## Project Overview

This repository contains a sample implementation of a Microservices Architecture using Docker, Docker Compose, Spring Boot, Angular, Spring Cloud, and Jenkins... The goal is to demonstrate how to containerize and orchestrate microservices, enabling scalability, flexibility, and ease of deployment.

## Features

- Dockerized microservices for easy deployment and scalability.
- Integration of Spring Boot for backend services.
- Angular for the frontend user interface.
- Spring Cloud for microservices coordination and discovery.
- Jenkins for continuous integration and automated builds.

## Technologies Used

- Docker
- Docker Compose
- Spring Boot
- Angular
- Spring Cloud
- Jenkins

## Getting Started

To get started with this project, you'll need to have Docker and Docker Compose installed on your machine. Clone this repository to your local environment.
### Build Docker images:
docker-compose build
### Run Docker containers:
docker-compose up -d
